# cutils #

**cutils** is a set of utils for C programs.

## List ##

##### STRUCTURE #####

```C
typedef struct item {
    CONTENT_TYPE content;
    struct item* next;
} item_t;
```

##### MACROS #####

- `LIST_APPEND(head,node)`: Appends a new item `node` into the list pointed by `head`.
- `LIST_APPEND(head,node)`: Appends a new item `node` into the list pointed by `head`.
